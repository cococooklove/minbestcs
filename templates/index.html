Chrome 확장 프로그램 설치 방법 — 최초 1회만
STEP 1
확장 프로그램 다운로드
상단 확장 프로그램 다운로드 버튼 클릭
→ zip 파일 저장 후 압축 해제
STEP 2
Chrome 확장 관리 열기
Chrome 주소창에 입력:
chrome://extensions
→ 오른쪽 상단 개발자 모드 토글 켜기
STEP 3
폴더 불러오기
압축해제된 확장 프로그램을 로드합니다 클릭
→ 압축 해제한 폴더 선택
→ 확장 프로그램 목록에 추가됨
STEP 4
수집 시작
네이버 셀러센터에 로그인
→ Chrome 우측 상단 확장 아이콘(🧩) 클릭
→ 리뷰 수집 시작 버튼 클릭
※ 설치 후에는 매번 STEP 4만 반복하면 됩니다.
